# Slooped Up Explorer

A mod for Satisfactory that significantly boosts Explorer speed and handling.

## Description

One pioneer accidentally drove an Explorer into an Alien Power Augmenter, and to their surprise, it didn’t explode! From this research, FICSIT Inc. has issued immediate upgrades for all Explorer models, increasing speed, improving handling, enabling manual control, and adding front hydraulics.

**NOTE**: The core functionality of the mod is complete, with the upgrades for the Explorer always enabled. More is planned such as sloop flames coming out of the exhaust and improved engine sounds.

## Compatibility

This mod is perfectly safe to uninstall at any time as all changes are made when each game session starts. Fully multiplayer compatible and works in dedicated servers.
